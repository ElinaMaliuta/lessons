4. Підключаємо 2 родини шрифтів з кількома стилями (font-style, font-weight) різними методами.

Підключення шрифту через імпорт з Google.

В папці CSS створюємо новий файл "fonts.css". Підключаємо його в файл "index.html" в тег "head" наступним рядком:
link rel="stylesheet" href="css/font.css"
Знаходимо в Google шрифт Montserrat. Копіюємо код для імпорту цього шрифта: @import url('https://fonts.googleapis.com/css2?family=Montserrat&display=swap'). Вставляємо код з буферу в файл fonts.css. Перевіряємо, чи підключився шрифт: призначаємо для якогось тегу (для bоdy, наприклад) родину шрифта.

Підключення шрифту через додання його в папку проекту.

Знаходимо шрифт на Google fonts або на сторонньому ресурсі. Скачуємо потрібний шрифт. В нашому випадку скачаний шрифт Caveat. Файли шрифта можуть бути з різними форматами: .woff2 та .woff або .ttf. В папку fonts необхідно додати або файли .woff2 та .woff або .ttf. Далі підключаюємо шрифти в файл "fonts.css" за допомогою @font-face. Файли з розширенням .woff2 та .woff підключаються по черзі: спочатку .woff2, потім .woff з форматами відповідно woff2 та woff. Файл з форматом .ttf підключається з форматом truetype. Перевіряємо, чи підключився шрифт: призначаємо для якогось тегу (для bоdy, наприклад) родину шрифта.
